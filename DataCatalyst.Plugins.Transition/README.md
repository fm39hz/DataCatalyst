# DataCatalyst.Plugins.Transition

Transition and sensor condition data models. Uses NumericCompare for operator types.

## Types

```
Models/TransitionDef           — TargetState, Priority, Conditions, Influences
Models/ConditionGroupDef       — All (AND), Any (OR), None (NOT)
Models/SensorConditionDef      — Signal, Op, Value, ExitValue
Models/SensorInfluenceDef      — Signal, Weight
```
