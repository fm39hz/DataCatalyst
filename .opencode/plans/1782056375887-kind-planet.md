# Concept Gộp Vào Core — Phân Tích & Đề Xuất

## Hiện trạng

```
Core: [DataComponent] Concept { string[] Value }          ← field đã ở Core
Plugin: ConceptRegistry + ConceptCatalog + GameConceptPlugin  ← system còn ở plugin
```

## Dependency Graph

```
ConceptRegistry → System.* (zero DC deps)          ✅ move to Core
ConceptCatalog<T> → DataEntry (Abstractions)       ✅ move to Core  
ConceptKind → System.* (empty structs)              ✅ move to Core
DataConceptAttribute → System.* (zero DC deps)       ⚠️ move to Core (update SourceGen string)

GameConceptPlugin → Core + ConceptRegistry         ❌ keep plugin (is ICatalogPlugin)
DataCatalogConceptExtensions → Core + plugin        ❌ keep plugin (retrieves GameConceptPlugin)

ConceptsFromDataGenerator → Roslyn (no DC deps)    ❌ keep SourceGen (string refs → update)
ConceptGenerator → Roslyn (no DC deps)              ❌ keep SourceGen (string refs → update)
```

## Đề xuất

```
CORE (sau khi gộp):
  WellKnownComponents → Concept struct (đã có)
  ConceptRegistry     → runtime name↔type mapping   ← mới
  ConceptCatalog<T>   → typed scoped access         ← mới
  ConceptKind         → StateConceptKind, SensorConceptKind  ← mới
  DataConceptAttribute → [DataConcept("name")]      ← mới

PLUGIN (giữ nguyên):
  GameConceptPlugin        → ICatalogPlugin hook
  DataCatalogConceptExtensions → GetConcept<T>() extension

SOURCEGEN (cập nhật string refs):
  ConceptsFromDataGenerator → "DataCatalyst.Core.ConceptRegistry"
  ConceptGenerator          → "DataCatalyst.Core.DataConceptAttribute"
```

## Separation of Concern

| Layer | Có gì | Việc |
|-------|-------|------|
| **Core** | Concept field, ConceptRegistry, ConceptCatalog, ConceptKind | Data model + runtime access |
| **Plugin** | GameConceptPlugin, GetConcept extension | Pipeline hook + consumer API |
| **SourceGen** | ConceptGenerator, ConceptsFromDataGenerator | Generate constants từ data |

## Lợi ích

1. **Concept không cần plugin** — `GetField<string[]>()` đã có sẵn
2. **Plugin nhẹ** — chỉ còn hook + extension method
3. **SourceGen string refs** — 1 lần cập nhật, đơn giản
4. **Không circular deps** — Core không phụ thuộc plugin
5. **Backward compatible** — SourceGen vẫn generate Concept structs
